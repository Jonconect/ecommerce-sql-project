# ecommerce-sql-project
Modelagem lógica de banco de dados para e-commerce com foco em regras de negócio complexas, como distinção entre clientes PF/PJ e múltiplos métodos de pagamento. Inclui scripts SQL (DDL/DML) e consultas analíticas avançadas (Joins, Having, Group By) para extração de insights estratégicos de vendas e logística. Desafio de projeto DIO.
